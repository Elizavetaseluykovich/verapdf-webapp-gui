What does this application do?

Validates PDF documents against a family of standards addressing accessibility: WCAG and PDF/UA.

Complete set of machine checks which can be performed by deterministic algorithms and do not require human interaction.

An experimental profile that covers human checks for correctness of semantics.

How to use this application?

Upload your PDF document, choose the validation profile and start validation.

Get the report with the list of errors and visual preview of the related content.

Inspect the corresponding tag(s) in the document structure tree.
